## Project Overview
# Student Academic Monitoring System
A web-based Student Academic Monitoring System developed using PHP Native and MySQL. The system enables administrators and homeroom teachers to manage student academic data, attendance records, and grade reports, while allowing parents to securely monitor their children's academic progress through an access code verification mechanism.
This project was developed as an undergraduate thesis project and demonstrates skills in backend development, database management, role-based access control, and academic information systems.

## Features
- Multi-user authentication (Admin, Teacher, Parent)
- Student attendance monitoring
- Student grade management
- Parent access code verification
- Search and filtering by class and subject
- WhatsApp quick link to teachers
- Grade calculation and evaluation
- Attendance and grade report export to Excel
- Role-based dashboard for different users
- CRUD management for teachers, students, classes, and subjects

## Installation
1. Clone the repository
```bash
git clone https://github.com/yunanda0050/student-academic-monitoring-system.git
```
2. Import the MySQL database.
3. Configure database connection settings.
4. Run Apache and MySQL using XAMPP.
5. Open the application in your browser.
